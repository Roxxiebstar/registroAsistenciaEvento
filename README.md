# registroAsistenciaEvento
Esta página web gestiona el registro de los jóvenes de la Iglesia Manantial, sede Kennedy para el evento Maranatha, que se realiza anualmente. Permite validar las inscripciones y contactar posteriormente a quienes no puedan asistir.
# Documentación breve 
--En la prueba técnica desarrollé un sistema de registro para el evento Maranatha de la Iglesia Manantial, sede Kennedy, dirigido a los jóvenes. La interfaz busca ser atractiva y significativa, y el sistema incluye validaciones para que los usuarios completen correctamente los campos, asegurando que los correos y demás datos no queden vacíos. La página muestra dos tablas, una con los registrados y otra con el total de inscritos, incluyendo quienes no podrán asistir, facilitando un control organizado. Además, se realizaron pruebas unitarias en los módulos de correos, nombres y opciones “Sí/No” para garantizar que cada componente funcionara correctamente antes de integrarse al sistema completo.--
